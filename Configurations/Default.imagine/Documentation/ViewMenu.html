The View Menu

The View menu allows editing the displaying style of the documents currently selected, and refreshing the display.

Edit Fonts

Edit the displaying font, font size, and color of the document currently selected. These settings are independent for textual content and XML tags. This is in favor of making content easier to distinguish from markup. The default values for these settings are accessible through Preferences in the Window menu.

Refresh

Refresh the display of the document currently selected.

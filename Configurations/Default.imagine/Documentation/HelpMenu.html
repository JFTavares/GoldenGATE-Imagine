The Help Menu

The Help menu provides access to help content on all aspects and parts of the GoldenGATE editor, including plug-ins. The main help topics are directly accessible.

Help on <Topic>

This function opens the Help window or brings it to the front, and switches the chapter on <Topic>.

Help

This function opens the Help window or brings it to the front, not dealing any changes to the chapter currently displayed.

About

This shows a little About GoldenGATE dialog.
